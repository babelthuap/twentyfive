Can you find: five five-letter words with twenty-five unique letters?

Inspired by this Matt Parker video. This solution uses 100% vanilla JavaScript (i.e., no libraries) and runs entirely in your browser. I use three main "tricks" to make it faster:

  • represent each word as a 32-bit int and use bitwise AND to determine if two words have letters in common
  • only consider words that don't have letters in common with the words already chosen
  • use web workers for parallelism

Source code: https://github.com/babelthuap/twentyfive
